# Product Overview

Patience is a native macOS application built with Swift and SwiftUI for testing system resilience and performance under various conditions.

## Purpose

A comprehensive testing framework that provides:
- **Live Testing**: Real-time system monitoring and stress testing
- **Log Analysis**: Intelligent parsing and analysis of system logs
- **Adversarial Testing**: Automated testing with challenging scenarios
- **Reporting**: Detailed performance and reliability reports

## Key Features

- Native macOS GUI built with SwiftUI
- Real-time performance monitoring
- Automated test scenario generation
- Comprehensive reporting and analytics
- Extensible testing framework

## Target Users

- System administrators
- DevOps engineers
- Quality assurance teams
- Performance testing specialists
